Presets¶

You can create presets for:

  • Hair Deformers
  • Single Hair System
  • Multiple Hair Systems

The way it works is simple: it will store required node properties and inputs. It won't store meshes, textures, vertex groups etc. For that you would use Hair Library.

Deformer Presets¶

Example of Color Set deformer and applying few example presets to it

Hair System Presets¶

Example of Strands from Grid Guide Hair System presets (base, breakup and then flyaway layers). First add hair system, then you can pick preset that will be applied to it

Multiple Hair System Presets¶

They work almost the same as Single Hair System presets, but they store all the hair systems assigned to object. Again it won't store meshes, textures, vertex groups etc. For that you would have to use Hair Library.
