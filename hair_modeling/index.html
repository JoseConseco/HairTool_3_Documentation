Hair Modeling¶

Hair Modeling works in object mode. You have to select 'Hair Modeling' tool from left sidebar(T)

The main reasons for these Modeling Tools being developed for object mode, is to avoid default messy drawing of points. See image below (edit mode):

Drawing curves in editmode in Blender 3.6, is not as bad is it was for old curves (there was no depth culling). Still you may find it easier to work with Hair Modeling strands overlay

You can deform hair curves, in natural way, using 'Chain' mode (IK): it will move hair endings, while preserving hair curve length. There is 'Normal' mode which works same as blender proportional editing.

This video is a bit outdated (for HT 2.46), but the functionality in Hair Tool 3.0 is mostly the same (except it works only on new hair curves now!)

There are also some basic helper tools:

  • duplicate splines
  • delete/cut splines
  • deform splines using Linear Deformer
  • adjust tilt and taper

Showcasing Chain (IK) transform mode (G,R,S keys), with Hair Modeling tool

Hotkeys¶

Note: Switching to Hair Modeling Tool will override some default blender 3d view shorcuts. To select other objects use Ctrl+RMB In round brackets I marked optional modifier keys eg. (Alt).

Hotkey Function
Ctrl+MMWheel Offset selection
(Shift)+RMB Select Points
(Alt)+A (de)Select All
(Alt)+B+LMB Box (de)Select
(Alt)+C Circle (de)Select
Ctrl+I Invert Selection
G, R, S Transform spline
Ctr+(Alt)+LMB Lasso (de)Select
X, Del Delete Selected
Ctrl+D Linear Deformer
Shift+D Duplicate strands
Alt+S Adjust Radius
Alt+R Reset Radius
Ctrl+T Adjust Tilt
Alt+T Reset Tilt

Offset selection¶

Ctrl + MMB scroll - changes which point along the strand length can be selected

Linear Deformer¶

Ctrl+D - rotate, bend, scale, move selected curves using on screen gizmo (special thanks to Mira Tools developer for letting me include modified version of his tool in HTool)

Extend¶

E - adjust length of selected strands

Duplicate and Delete¶

Shift+D - duplicate; X or Del delete selected strands

Knife¶

K - cut selected strands. Define cut line using LMB (first to mark start point, then next LMB click to mark end point)

Edit Mode¶

Blender 3.6 does not have basic modeling tools for new hair curves. Few of them were added to Hair Tool.
