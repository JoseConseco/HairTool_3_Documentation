Hair Setup For Baking¶

The baking panel is located right Sidebar(N) -> "Hair Tool" tab -> 'Hair Bake' Panel

To Hair Tool comes in with baking blend file template, that makes it easy to bake various hair maps. There are two options to bake your textures:

  • Open baking template file, and bake your hair setup there (save your work before opening baking file)
  • Bake your hair setup directly in your character file - using 'Import Baking Scene' button - this will append 'BakingHair' scene to your file

Baking Template consist of few Hair Systems attached to meshes. By tweaking these Hair Systems, you can get desired result.

See Hair System - to learn how they works

Note: To get proper bake, template file uses Round Profile type.

This file has also curve, with PreviewBakeMat material assigned to its profile. After pressing 'Bake', the output textures will be automatically applied to this material, so you can see result directly in same file.
