Strands From Profile¶

It uses Strands from Grid Surface Hair Generator, to generate hair strands from active object profile. This can be used in recursive way, to generate hair cards based on other hair cards. Only for Hair Tool 4.2 and up.

Location¶

Select hair, with mesh profile and use:

  • Ctrl+Shift+H > Strands from Profile (as guide)

In operator properties popup (F9) you can select if base hair mesh should be rendered in 'Wire' mode or kept it as is.

Examples¶

Basic example of using 'Strands from Profile' executed on hair-card object, that used Hair System with Flat Profile type. At any time you can go to base object, modify it and new generated strands will follow it

Example of using 'Strands from Profile' on braid, to add more detail to it. I had to bump up strands subdivisions up to 5 to have enough resolution on them to follow braid

Parameters¶

These are the same as in Strands from Grid Surface generator. The only extra parameter is option, in operator properties popup (F9), to select if base hair mesh should be rendered in 'Wire' or 'Solid' mode. For braid above I used Solid mode, so we can see both haircard meshes at the same time.
